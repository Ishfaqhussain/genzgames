Visual Perception Research Study
This is a web-based instrument designed to collect data for a research paper on **Gen Z vs. Millennials: AI Detection Capabilities**.

Features
Video Playback: Custom HTML5 video player.
Timer Logic: JavaScript captures "Time to Decision" (ms) for cognitive load analysis.
Data Collection: Serverless submission to Google Sheets via Google Apps Script.

Tech Stack
- HTML5 / CSS3 / Vanilla JS
- Hosted on Netlify
- Backend: Google Sheets API
